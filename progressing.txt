[~] FIDO-093 — Supporto passkey e chiavi di sicurezza FIDO2 come metodo MFA
------------------------------------------------------------------------------
  Priorita: MEDIA
  Dipendenze: Nessuna

  DESCRIZIONE:
  Aggiungere il supporto WebAuthn come terzo metodo MFA, accanto a TOTP e SMS.
  Gli utenti potranno registrare passkey (salvate in password manager come
  Bitwarden, 1Password, iCloud Keychain, Windows Hello) e chiavi di sicurezza
  hardware FIDO2 (YubiKey, Titan, SoloKeys) per l'autenticazione a due fattori.

  DETTAGLI TECNICI:
  Dipendenze npm:
    - @simplewebauthn/server (server workspace)
    - @simplewebauthn/browser (client workspace)
  Variabili ambiente (.env.example):
    - WEBAUTHN_RP_ID (default: localhost)
    - WEBAUTHN_RP_ORIGIN (default: http://localhost:3000)
    - WEBAUTHN_RP_NAME (default: Remote Desktop Manager)
  Prisma schema:
    - Nuovo modello WebAuthnCredential
    - Campo webauthnEnabled Boolean su User
    - Nuovi AuditAction: WEBAUTHN_REGISTER, WEBAUTHN_REMOVE, LOGIN_WEBAUTHN
  Backend:
    - webauthn.service.ts, webauthn.controller.ts, webauthn.routes.ts
    - auth.service.ts, auth.controller.ts, auth.routes.ts updates
    - config.ts: sezione webauthn
  Frontend:
    - webauthn.api.ts, auth.api.ts updates
    - WebAuthnSection.tsx, SettingsDialog.tsx, LoginPage.tsx updates

  File coinvolti:
