------------------------------------------------------------------------------
  Priorita: ALTA
  Dipendenze: CTX-003 (completato)
  Stato backend: COMPLETO (API sharing con re-cifratura credenziali)
  Stato frontend: PARZIALE (ShareDialog.tsx esiste ma non collegato)

  DESCRIZIONE:
  Aggiungere voce "Share" nel context menu per condividere connessioni
  con altri utenti. Due livelli di permesso:
    - READ_ONLY (View): il destinatario puo' solo aprire la connessione,
      NON vede la password ne' puo' modificare le impostazioni
    - FULL_ACCESS (Edit): il destinatario puo' anche modificare la
      connessione e visualizzare le credenziali decifrate

  DETTAGLI TECNICI:
  Backend (gia' esistente):
    - POST   /api/connections/:id/share         → Condividi (email, permission)
    - DELETE /api/connections/:id/share/:userId  → Revoca condivisione
    - PUT    /api/connections/:id/share/:userId  → Aggiorna permesso
    - GET    /api/connections/:id/shares         → Lista condivisioni
    Nota: entrambi i vault (owner + target) devono essere sbloccati per
    condividere, perche' le credenziali vengono ri-cifrate con la chiave
    del destinatario.

  Frontend DA IMPLEMENTARE:
    1. Aggiungere voce "Share" nel context menu (ConnectionTree.tsx)
       - Icona ShareIcon, disabilitata se !isOwner
    2. Collegare ShareDialog.tsx all'apertura dal context menu
       - Passare connectionId come prop
       - ShareDialog gia' ha: input email, select permesso, lista share
    3. In MainLayout.tsx: gestire stato apertura ShareDialog
       - Nuovo state: shareTarget (ConnectionData | null)
       - Prop onShare passata a ConnectionTree
    4. Migliorare ShareDialog:
       - Mostrare descrizione chiara dei permessi (tooltip/help text)
       - READ_ONLY: "L'utente potra' solo connettersi"
       - FULL_ACCESS: "L'utente potra' connettersi e modificare"

  FILE COINVOLTI:

       - Leggere navigator.clipboard.readText() su focus/input del display
       - Inviare al remote via Guacamole.OutputStream + Guacamole.StringWriter

       di default se non ci sono flag disable-copy/disable-paste)

       - Ctrl+Shift+C → navigator.clipboard.writeText(terminal.getSelection())

    2. Nessun pacchetto aggiuntivo necessario
