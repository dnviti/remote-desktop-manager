[~] NOTIF-015 — Notifiche in-app per condivisioni
------------------------------------------------------------------------------
  Priorita: BASSA
  Dipendenze: SHR-005

  DESCRIZIONE:
  Notificare l'utente quando qualcuno condivide una connessione con lui.

  DETTAGLI TECNICI:
    1. Modello Prisma:
       Notification { id, userId, type (enum), message, read Boolean,
                      relatedId, createdAt }
    2. Creare notifica quando shareConnection() viene chiamata
    3. Real-time: emettere evento Socket.IO "notification:new" al target
    4. Frontend:
       - Badge con conteggio notifiche non lette nell'AppBar
       - Popover/drawer con lista notifiche
       - Click su notifica naviga al contesto (es. connessione condivisa)
       - Endpoint: GET /api/notifications, PUT /api/notifications/:id/read

  FILE COINVOLTI:
    CREARE:  server/src/services/notification.service.ts
    CREARE:  server/src/routes/notification.routes.ts
    CREARE:  client/src/components/Layout/NotificationBell.tsx

    MODIFICARE: server/src/services/sharing.service.ts (trigger notifica)
    MODIFICARE: server/src/socket/index.ts (namespace notifiche)
    MODIFICARE: client/src/components/Layout/MainLayout.tsx (badge)
